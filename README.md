# Standoff Damage Counter

## Documentation

### Features
- **Real-time Damage Tracking**: Keeps track of damage dealt in real-time.
- **Weapon Customization**: Ability to customize weapon damage values.
- **User-Friendly Interface**: Simple and intuitive UI.
- **Data Export**: Export damage statistics and weapon configurations.

### How to Use
1. Clone the repository using:
   ```bash
   git clone https://github.com/daniilaskevic714-max/standoff-damage-counter.git
   ```
2. Navigate to the project directory:
   ```bash
   cd standoff-damage-counter
   ```
3. Run the application to start tracking damage.

### Weapons List
- Scar-L: 45 Damage
- AK-47: 50 Damage
- M4A1: 48 Damage
- AWP: 100 Damage
- USP: 25 Damage

### Neon Design Description
The application's design incorporates neon elements, giving it a modern and edgy feel. Bright neon colors are used for highlights and buttons, creating a visually engaging experience. This design choice aims to reflect the high-energy environment of gaming, enhancing the overall user interaction with the damage counter.
